# 快速开始

## 安装

1. 前往 [GitHub Releases](https://github.com/zexadev/jdnotes/releases) 下载最新安装包
2. 运行安装程序，按提示完成安装
3. 启动 JD Notes

## 配置 AI

1. 打开设置（左下角齿轮图标）
2. 在 AI 配置区域添加来源
3. 填写 API Key 和模型信息
4. 可添加多个来源，随时切换

## 开始写作

- 点击左侧「新建笔记」创建笔记
- 支持 Markdown 快捷语法
- 选中文本可唤出 AI 气泡菜单
- 右侧 AI 助手可进行对话
