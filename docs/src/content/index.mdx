# JD Notes

简洁高效的本地笔记应用，基于 Tauri + React 构建。

## 特性

- 本地优先，数据完全存储在你的设备上
- 内置 AI 助手，支持多种模型（OpenAI、Claude、Gemini、Ollama）
- 富文本编辑器，支持 Markdown
- 轻量快速，基于 Tauri 原生窗口

## 快速开始

前往 [GitHub Releases](https://github.com/zexadev/jdnotes/releases) 下载最新安装包。
