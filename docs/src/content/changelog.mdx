## v1.6.0

### 新增功能
- **表格功能**：支持插入表格、调整列宽，光标在表格内时显示增删行列的气泡菜单
- **AI 联网能力**：AI 助手可搜索互联网和读取网页内容（DuckDuckGo 搜索 + 网页抓取）
- **IP 定位工具**：AI 可获取用户大致地理位置
- **Responses API 支持**：新增 OpenAI Responses API provider，支持 Azure AI Foundry、GPT-5 等
- **MCP 数据库变更通知**：MCP 写入笔记后前端实时刷新列表

### 优化
- 移除查看/编辑模式切换，笔记打开即可编辑
- AI system prompt 注入当前时间，明确联网能力说明
- 表格操作重构为气泡菜单，交互更直观
- 仪表盘图表优化

### 修复
- 修复重试和编辑消息时旧对话仍发给模型的问题
- 修复搜索结果为空的问题
- 修复表格扩展导入方式

## v1.5.0

### 新增功能
- **AI Tools 函数调用**：AI 助手可通过工具按需读取、搜索、创建、修改和删除笔记，不再把笔记内容塞进提示词
- **多对话历史**：每个笔记支持多个独立对话，可新建、切换、删除
- **图片发送**：聊天中可附加图片发送给 AI（多模态支持）
- **MCP 自动注册扩展**：启动时自动注册到 9 个 AI 工具（Claude Code/Desktop、Cursor、Windsurf、Gemini CLI、Kiro、Copilot CLI、OpenCode、Cline）

### 优化
- AI 消息用 parts 结构渲染，tool calls 穿插在文字流中显示
- 用户消息改为右对齐气泡样式
- 模型选择器移至输入框上方（类似 Cursor）
- Tools 返回结果不再包含无意义的 ID

## v1.4.0

### 新增功能
- **MCP Server 集成**：内置 HTTP MCP Server，Claude Code 等 AI 编程工具可直接将内容保存到笔记
  - 支持创建笔记、追加内容、修改笔记三个工具
  - 启动时自动注册到 Claude Code，无需手动配置
- **待办列表**：支持创建待办事项，Markdown 兼容 `- [ ]` / `- [x]`
- **编辑器固定工具栏**：文本格式、列表、引用、代码块、插入图片等常用操作
- **图片功能**：支持工具栏插入、粘贴、拖拽插入，可缩放和预览大图
- **Ctrl+Click 打开链接**：类似 VS Code 的链接交互体验
- **AI 内联模式**：重构 AI 内容插入为编辑器内联模式
- **主题切换动画**：日夜动画开关替换原有主题按钮

### 优化
- 编辑器段落间距和待办列表对齐优化
- 图片显示效果和操作体验优化
- Ctrl+Click 链接视觉反馈

### 修复
- 修复复制链接格式和自动标题 API 路径问题
- 修复 Ctrl+Click 无法打开链接的问题
