# 功能介绍

## 富文本编辑器

基于 TipTap 的现代编辑器，支持：
- Markdown 快捷输入
- 标题、列表、代码块、引用等
- 斜杠命令快速插入内容

## AI 辅助

- 多模型支持：OpenAI 兼容、Anthropic Claude、Google Gemini、Ollama
- 多来源配置：同时保存多个 AI 来源，一键切换
- 气泡菜单：选中文本即可改进、翻译、总结
- AI 对话：右侧聊天面板，结合笔记上下文对话

## MCP Server 集成

- 内置 HTTP MCP Server，启动时自动运行
- 自动注册到 Claude Code、Cursor、Windsurf 等 9 个 AI 工具
- 6 个工具：读取（get_note、search_notes、list_notes）+ 写入（create_note、append_note、update_note）
- Agent Skill 自动安装：启动时自动安装到 Claude Code、Copilot、Gemini CLI

## 数据管理

- 本地 SQLite 存储，数据完全离线
- 支持自定义存储位置
- JSON 导入导出
- 选择性导出笔记

## 提醒通知

- 笔记内设置提醒时间
- 系统通知弹窗提醒
